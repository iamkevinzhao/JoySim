## JoySim -- A Simple Multi-Sensor Simulator

#### Overview

JoySim is a simple multi-sensor simulator for verifying sensor-fusing algorithms such as Kalman Filters and the Covariance Project Filter. 

#### Dependencies

- Qt 5.9
- C++11
- CMake 2.8.1

#### Version

- v1.0.0
